🚀 GitHub Issue 最新コメントジャンプ

GitHub Issueの最新コメントに一瞬でジャンプするブックマークレットです。 コメント数が多い Issue でも、クリック1回で最新コメントへ移動できます。

📌 ブックマークレット

下のボタンをブックマークバーにドラッグ＆ドロップしてください：

⚡ 最新コメントへジャンプ

📖 使い方

  1. 上の「⚡ 最新コメントへジャンプ」ボタンをブックマークバーにドラッグ＆ドロップ
  2. GitHub Issueページを開く
  3. ブックマークバーの「最新コメントへジャンプ」をクリック
  4. 自動的に最新コメントまでスクロール＆ハイライト表示
javascript:(function(){ var comments = document.querySelectorAll('.timeline-comment'); if (comments.length > 0) { comments[comments.length-1].scrollIntoView({ behavior: 'smooth', block: 'center' }); // 最新コメントを一時的にハイライト comments[comments.length-1].style.backgroundColor = '#fffbeb'; setTimeout(function(){ comments[comments.length-1].style.backgroundColor = ''; }, 2000); } else { alert('コメントが見つかりませんでした'); } })();

🔧 その他の方法

方法1: キーボードショートカット 最速

Ctrl+F → 今日の日付を検索（例: 2025-10-02）→ 下矢印で最新まで移動

方法2: GitHub CLI

gh issue view 1 --repo Tenormusica2024/Private --comments | tail -50

方法3: URL直接アクセス

最新コメントのURL（#issuecomment-XXXXXX）を直接ブラウザに入力

方法4: ブラウザ開発者ツール

// F12 でコンソール開いて実行 document.querySelectorAll('.timeline-comment')[ document.querySelectorAll('.timeline-comment').length - 1 ].scrollIntoView({behavior: 'smooth'});
